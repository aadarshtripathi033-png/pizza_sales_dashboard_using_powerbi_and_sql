# pizza_sales_dashboard_using_powerbi_and_sql
##  Pizza Sales Analysis using SQL & Power BI

###  Project Overview

This project focuses on analyzing pizza sales data to extract meaningful business insights using **SQL** for data processing and **Power BI** for data visualization. The objective is to understand customer behavior, sales trends, and product performance to support data-driven decision-making.

###  Objectives

* Analyze overall sales performance
* Identify best-selling and worst-selling pizzas
* Understand customer ordering patterns
* Evaluate revenue trends over time
* Generate insights to improve business strategy

###  Tools & Technologies Used

* SQL (Structured Query Language) – Data extraction, transformation, and analysis
* Power BI – Data visualization and dashboard creation
* Dataset – Pizza sales transactional data

###  Key Features & Analysis

1. Sales Performance Analysis

* Total revenue generated
* Total number of orders
* Average order value

2. Product Analysis

* Top-selling pizzas by quantity and revenue
* Least popular pizzas
* Category-wise sales performance

3. Time-Based Analysis

* Daily, weekly, and monthly sales trends
* Peak order hours and busiest days

4. Customer Behavior Insights

* Ordering patterns
* Frequency of orders
* Preferred pizza categories

## Dashboard Highlights (Power BI)

* Interactive dashboards with filters and slicers
* KPI cards for quick insights
* Sales trends visualization (line charts, bar charts)
* Category-wise and size-wise analysis

### Key Insights

* Identified peak sales hours to optimize operations
* Discovered most profitable pizza categories
* Found low-performing products that may need improvement or removal
* Helped in better inventory and marketing decisions

### Conclusion

This project demonstrates how raw data can be transformed into actionable insights using SQL and Power BI. It highlights the importance of data analytics in improving business performance and strategic planning.

### Future Enhancements

* Add predictive analysis using Machine Learning
* Integrate real-time data updates
* Enhance dashboard with advanced DAX calculations

### How to Use

1. Load dataset into SQL database
2. Run SQL queries for analysis
3. Import cleaned data into Power BI
4. Explore the interactive dashboard
